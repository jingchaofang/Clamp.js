#clamp

clamp.js在掌阅书城的应用及扩展

```less
// 单行溢出打点...
.ellipsis() {
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
}
// 多行溢出打点...
.multi-ellipsis(@line:2) {
    overflow: hidden;
    display: -webkit-box;
    -webkit-box-orient: vertical;
    text-overflow: ellipsis;
    -webkit-line-clamp: @line;
}
```
